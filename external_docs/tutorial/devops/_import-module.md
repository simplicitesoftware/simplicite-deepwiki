Manual Module Import
--------------------

1. Log in with **ADMIN** rights.
2. From the left sidebar, navigate to **Project > Modules**.

   ![Project Modules Navigation](img/module/import_module1_.png)

3. In the list of modules, click the row corresponding to the module you want to update.

   ![Select Module](img/module/import_module2_.png)

4. In the form, locate the **File** field and the three icons next to the input field.
5. Click the second icon (**Edit**), then click **Choose File**.

   ![Choose File](img/module/import_module3_.png)

6. Select the module file from your computer.
7. Save the form by clicking the **Save** button in the header.

   ![Save Form](img/module/import_module4_.png)

8. In the **Import/Export** panel on the right side of the form, click **Import Module**, then confirm by clicking **Yes**.

   ![Import Module Step 1](img/module/import_module5_.png)
   ![Import Module Step 2](img/module/import_module6_.png)

9. Wait for the import process to complete.

   :::tip Shortcut
   You can open the cache menu quickly at any time using the keyboard shortcut **<kbd>Alt</kbd> + <kbd>C</kbd>**.
   :::

10. Access the **Clear Cache** menu, either using the shortcut icon in the top-left header or pressing **<kbd>Alt</kbd> + <kbd>C</kbd>**.

    ![Clear Cache Shortcut](img/module/import_module6_.png)

11. Click **Clear all sessions and server caches**.

    ![Clear All Caches](img/module/import_module7_.png)
